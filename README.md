# Moota.co WHMCS Payment Gateway Module

## Dev Install ##
- `composer install`
- `copy` or symlink everything inside `modules/gateways/` into your WHMCS' 
  `modules/gateways/`

## Minimum Requirements ##

- PHP 5.6
